## W4 · LLM Tool-Calling Agent（进行中）

**目标**：live demo 场景——用户用自然语言提问 → Agent 自动编排多工具调用 → 带来源引用的分析结论。

**完成判据**（三条全绿即为 W4 结束）：
1. 20 个测试 query 的 `task_completion_rate ≥ 0.80`
2. 平均 step 数 ≤ 4，p95 latency ≤ 8s
3. 前端演示 3 个预设复杂 query 零崩溃
